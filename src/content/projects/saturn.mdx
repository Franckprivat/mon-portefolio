## Le projet

Saturn est mon projet le plus ambitieux : un réseau social complet inspiré de Discord,
en versions web **et** mobile, construit de bout en bout — du schéma de base de
données à l'interface.

## Fonctionnalités clés

- **Chat en temps réel** avec Socket.IO : messages privés et de groupe, accusés de lecture, messages vocaux enregistrés dans le navigateur
- **Communautés** : serveurs avec salons textuels et vocaux, rôles et invitations
- **Appels audio / vidéo** en topologie mesh WebRTC, avec journal d'appels
- **Version mobile** en React Native partageant la même API NestJS
- **Thèmes personnalisables** : palettes pilotées par variables CSS, sans flash au chargement

## Ce que j'ai appris

Le temps réel change tout : gestion des reconnexions socket, synchronisation de l'état
entre plusieurs clients, signalisation WebRTC… C'est aussi sur ce projet que j'ai
structuré un vrai monorepo backend / frontend avec Docker et Nginx en reverse proxy.
