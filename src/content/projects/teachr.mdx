## Le projet

Teach'r est une application web fullstack dédiée à l'apprentissage, développée de
novembre 2024 à janvier 2025.

## Fonctionnalités

- **Authentification complète** : inscription et connexion sécurisées
- **Interface utilisateur intuitive** construite avec React et Tailwind CSS
- **Gestion dynamique des données** via une API Symfony connectée à une base MariaDB
- **Environnement conteneurisé** avec Docker pour un setup reproductible

## Ce que j'ai appris

Mon premier vrai projet fullstack avec un backend PHP : la structure de Symfony
(entités, contrôleurs, sérialisation) et le dialogue entre un front React et une
API REST testée sous Postman.
