## Le projet

Ce portfolio est lui-même un projet : je voulais un site rapide comme une page statique,
mais vivant comme une application moderne.

## Choix techniques

- **Astro** pour le rendu statique et les Content Collections (projets et articles en MDX)
- **Tailwind CSS v4** avec un design system en variables CSS — le thème clair/sombre bascule sans flash
- **GSAP + ScrollTrigger** pour les révélations au scroll, et **Lenis** pour le smooth scrolling
- **Zéro framework côté client** : les interactions (palette de commandes, curseur, filtres) sont en TypeScript vanilla

## Détails auxquels je tiens

Le score Lighthouse, l'accessibilité (navigation clavier, `prefers-reduced-motion` respecté),
et une palette de commandes ouverte avec `Ctrl + K` pour naviguer et chercher les projets.
